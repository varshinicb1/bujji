# CLI Reference

## Usage

```bash
bujji [OPTIONS] COMMAND [ARGS]...
```

## Commands

### `bujji chat`

Start an interactive chat or send a single message.

```bash
bujji chat "What does this project do?"
bujji chat --stream "Explain the architecture"
```

### `bujji plan`

Create a structured plan for a task.

```bash
bujji plan "Add user authentication"
```

### `bujji tools`

List all available tools with descriptions.

```bash
bujji tools
```

### `bujji doctor`

Run system diagnostics.

```bash
bujji doctor
```

### `bujji memory`

Show memory statistics.

```bash
bujji memory
```

### `bujji status`

Show BUJJI component status.

```bash
bujji status
```
